               buff_entries.append(BonusDiceEntry(formula=buff.bonus_value, label=buff.name))
   ```
   Append `buff_entries` to the existing `bonus_dice` list in the `RollRequest` constructor. The `RollService` already rolls `bonus_dice` entries and returns results in `AttackRollResult.bonus_dice_results` as `(formula, signed_total, label)` tuples.

   - For flat bonuses (no dice, just a number like "+2"): show `+ {label}({signed_total})` on first, `+{signed_total}` on subsequent (flat bonuses have no dice notation to abbreviate)
   - The `label` comes from `bonus_dice_results[i][2]` (third element of the tuple)

   for ar in attack_rolls:

   ```
   Use the existing `_color_damage_segment()` method and `DAMAGE_COLORS` dict. If only one damage type, skip the breakdown (redundant with total).
