**BUG-09 + BUG-10: Crit gold coloring and extra blank lines**

1. Find `_wrap_crit_line()` method. Currently it wraps content in a `<div>` with gold background. Change the `<div>` to `<span>`:
   ```python
   def _wrap_crit_line(self, html_content: str) -> str:
       return (
           f'<span style="background-color:rgba(212,175,55,0.25); '
           f'padding:1px 4px; border-radius:2px;">'
           f'{html_content}</span>'
       )
   ```
   This fixes BOTH BUG-09 (block-level `<div>` causes full-width gold background) and BUG-10 (`<div>` block element adds implicit paragraph margins in QTextEdit's HTML renderer).

2. Find `_wrap_miss_line()` method. Apply the same `<div>` to `<span>` change:
   ```python
   def _wrap_miss_line(self, html_content: str) -> str:
       return (
           f'<span style="background-color:rgba(180,0,0,0.18); '
           f'padding:1px 4px; border-radius:2px;">'
           f'{html_content}</span>'
       )
   ```
