Implement EquipmentService in `src/equipment/service.py` with the following methods:

**1. `scale_dice(base_dice: str, size: str) -> str`** (module-level helper)
- Parse "NdM" format by splitting on 'd'
- Multiply the count N by SIZE_DICE_MULTIPLIER[size]
- Return f"{scaled_N}d{M}"
- For "2d6" + Large: 2*2=4 → "4d6"

**2. `compute_weapon_action(weapon: WeaponData, magic_bonus: int, monster: Monster) -> dict`**
- Determine ability: if weapon.is_ranged and not weapon.is_thrown → DEX. If weapon.is_finesse and DEX_mod > STR_mod → DEX. Else STR.
- Use MonsterMathEngine to get DerivedStats (prof bonus and ability mods) OR compute directly: prof from _PROF_BY_CR table, ability_mod from (score-10)//2
- to_hit = ability_mod + prof_bonus + magic_bonus
- damage_dice = scale_dice(weapon.damage_dice, monster.size)
- damage_bonus = ability_mod + magic_bonus
- Return dict with: name, to_hit_bonus, damage_dice, damage_bonus, damage_type, is_equipment_generated=True

**NOTE on approach:** To avoid coupling to MonsterMathEngine (which takes a full Monster), compute prof_bonus and ability_mod inline using the same formulas. Import _PROF_BY_CR from engine if it's accessible, or duplicate the small lookup. The research shows MathValidator._pick_ability() exists but is private — reimplement the ability selection logic cleanly in EquipmentService.

**3. `compute_armor_ac(armor: ArmorData, magic_bonus: int, monster: Monster) -> dict`**
- dex_mod = (monster.ability_scores.get("DEX", 10) - 10) // 2
- If armor.dex_limit is None: ac_dex = dex_mod (light armor — full DEX)
- If armor.dex_limit == 0: ac_dex = 0 (heavy armor — no DEX)
- Else: ac_dex = min(dex_mod, armor.dex_limit) (medium armor — capped)
- total_ac = armor.base_ac + ac_dex + magic_bonus
- Return dict with: ac=total_ac, stealth_disadvantage=armor.stealth_disadvantage, str_requirement_met=(monster STR score >= armor.str_requirement), armor_name=armor.name, magic_bonus=magic_bonus

**4. `compute_shield_bonus(magic_bonus: int) -> int`**
- Return 2 + magic_bonus

**5. `compute_focus_bonus(focus_magic_bonus: int) -> dict`**
- Return dict with: spell_attack_bonus=focus_magic_bonus, spell_dc_bonus=focus_magic_bonus
