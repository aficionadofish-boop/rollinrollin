**src/persistence/service.py** — Create PersistenceService modeled on SettingsService:

```python
_FILENAMES = {
    "loaded_monsters": "loaded_monsters.json",
    "encounters": "encounters.json",
    "modified_monsters": "modified_monsters.json",
    "macros": "macros.json",
}
```

Constructor: `__init__(self, workspace_root: Path)` storing `self._root = workspace_root`.

Core private methods:
- `_load(self, category: str) -> dict | list` — Read JSON from `self._root / _FILENAMES[category]`. On missing file, return `{}` for modified_monsters, `[]` for others. On `json.JSONDecodeError` or `OSError`, return empty default (same as missing). Do NOT overwrite the corrupt file on load — only on next explicit save (per CONTEXT.md discretion: gives user one chance to recover).
- `_save(self, category: str, data: dict | list) -> None` — Write JSON with `indent=2` and `encoding="utf-8"`.

Public methods — one load/save pair per category:
- `load_loaded_monsters(self) -> list` — returns list of source file paths (strings)
- `save_loaded_monsters(self, data: list) -> None`
- `load_encounters(self) -> list` — returns list of `{name: str, members: [{name: str, count: int}]}`
- `save_encounters(self, data: list) -> None`
- `load_modified_monsters(self) -> dict` — returns `{name: {ability_scores, saves, ...}}`
- `save_modified_monsters(self, data: dict) -> None`
- `load_macros(self) -> list` — returns list (or single-element list with last-active buffer)
- `save_macros(self, data: list) -> None`

Utility methods:
- `flush(self, category: str) -> None` — Writes empty structure for category (`{}` for modified_monsters, `[]` for others). Does NOT delete the file.
- `flush_all(self) -> None` — Calls flush() for each category.
- `count(self, category: str) -> int` — Returns `len(self._load(category))`.
- `categories(self) -> list[str]` — Returns list of category keys.

All load methods delegate to `_load()`. All save methods delegate to `_save()`.
