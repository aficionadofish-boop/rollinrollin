    <implementation>
      Create src/parser/formats/fivetools.py with:

      CONSTANTS / REGEX (compile at module level, not inside functions):
        BLOCKQUOTE_PREFIX = re.compile(r'^>\s?', re.MULTILINE)
        BLOCK_START_RE = re.compile(r'^>#{1,2}\s+(.+)$', re.MULTILINE)
        TYPE_RE = re.compile(r'^\*(\w[^,*]+),\s*([^*]+)\*$')
        AC_RE = re.compile(r'\*\*Armor Class\*\*\s+(\d+)')
        HP_RE = re.compile(r'\*\*Hit Points\*\*\s+(\d+)')
        CR_RE = re.compile(r'\*\*Challenge\*\*\s+([\d/]+)')
        SAVES_RE = re.compile(r'\*\*Saving Throws\*\*\s+(.+)')
        SAVE_BONUS_RE = re.compile(r'(Str|Dex|Con|Int|Wis|Cha)\s*([+-]\d+)', re.IGNORECASE)
        ABILITY_CELL_RE = re.compile(r'\|(\d+)\s*\([+-]\d+\)')
        ABILITIES = ['STR', 'DEX', 'CON', 'INT', 'WIS', 'CHA']
        ACTION_SECTION_RE = re.compile(r'^#{1,3}\s+(Actions|Legendary Actions|Reactions|Bonus Actions|Traits)', re.IGNORECASE)
        ACTION_NAME_RE = re.compile(r'^\*{2,3}([^*]+?)\.\*{2,3}')
        TO_HIT_RE = re.compile(r'([+-]\d+)\s+to\s+hit', re.IGNORECASE)
        HIT_LINE_RE = re.compile(
            r'\*?Hit:\*?\s+(\d+)\s*\(([^)]+)\)\s+(\w+)\s+damage'
            r'(?:\s+plus\s+\d+\s*\(([^)]+)\)\s+(\w+)\s+damage)?',
            re.IGNORECASE
        )

      SEGMENTATION: _segment_blocks(content: str) -> list[tuple[str, str]]:
        Scan for all ">## Name" or ">### Name" lines using BLOCK_START_RE.finditer().
        For each match, the blockquote block runs from the match position to the next match
        or to the end of the blockquote region (line without ">" prefix).
        Return list of (name, block_text) tuples.
        Collect lore text: plain (non-">") paragraphs that follow each blockquote block.

      IMPORTANT — blockquote stripping before field extraction:
        strip_blockquote(text: str) -> str: Apply BLOCKQUOTE_PREFIX.sub('', line) per line.
        Always call strip_blockquote() on a block before applying AC_RE/HP_RE/CR_RE/SAVES_RE etc.

      FIELD EXTRACTION (each returns Optional value):
        _extract_ac(clean_text) -> Optional[int]
        _extract_hp(clean_text) -> Optional[int]
        _extract_cr(clean_text) -> Optional[str]
        _extract_type(clean_text) -> str  (first word after "*Size " in first italic line)
        _extract_ability_scores(clean_text) -> dict[str, int]  (ABILITY_CELL_RE.findall per line, zip with ABILITIES)
        _extract_saves(clean_text) -> dict[str, int]  (SAVES_RE then SAVE_BONUS_RE.findall)

      ACTION EXTRACTION: _extract_actions(clean_text) -> list[Action]:
        Split text at ACTION_SECTION_RE boundaries to get action block.
        Within action block, split on ACTION_NAME_RE to get individual action texts.
        For each action text:
          - Join adjacent lines within the action (handles *Hit:* on second line per Pitfall 6)
          - Apply ACTION_NAME_RE → name
          - Apply TO_HIT_RE → to_hit_bonus (int or None)
          - Apply HIT_LINE_RE.finditer() → damage_parts (see Pattern 4 in RESEARCH.md)
          - is_parsed = to_hit_bonus is not None and len(damage_parts) > 0
          - raw_text = original action text stripped
          - Create Action with all fields

      MAIN ENTRY: parse_fivetools(content: str) -> ParseResult:
        segments = _segment_blocks(content)
        if not segments: return ParseResult(monsters=[], failures=[], warnings=[])
        For each (name, block_text, lore_text) in segments:
          clean = strip_blockquote(block_text)
          Build Monster; incomplete = any(ac is None, hp is None, cr is None)
          Use sentinel defaults: ac=0 if None, hp=0 if None, cr="?" if None
          Append to monsters list
        Return ParseResult(monsters=monsters, failures=[], warnings=[])
