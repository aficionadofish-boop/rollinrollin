    <files>src/parser/statblock_parser.py, src/tests/test_parser_dispatch.py</files>
    <behavior>
      detect_format(content: str) -> str:
        - Content with `>- **Armor Class**` line → returns 'fivetools'
        - Content with `___` line AND `**Armor Class**` (no `>` prefix) → returns 'homebrewery'
        - Content with `## ` heading AND `**Armor Class**` bullet (no `>` prefix) → returns 'plain'
        - Content with none of the above → returns 'unknown'
        Detection is mutually exclusive and in priority order: fivetools > homebrewery > plain > unknown

      parse_file(path: Path) -> ParseResult:
        - Reads file with Path.read_text(encoding='utf-8')
        - On OSError or UnicodeDecodeError: returns ParseResult with one ParseFailure(source_file=str(path), monster_name="", reason=str(e)), monsters=[], warnings=[]
        - Detects format → dispatches to correct parse_* function
        - On 'unknown' format: returns ParseResult(monsters=[], failures=[], warnings=[f"Unrecognized format in {path.name}"])
        - Returns the ParseResult from the format parser as-is (failures from the sub-parser are preserved)

      ImportResult dataclass (in src/parser/statblock_parser.py or src/parser/models.py — put it in models.py):
        ImportResult(filename="goblin.md", success_count=2, incomplete_count=1, failures=[ParseFailure(...)])
        - success_count: int (monsters parsed without error)
        - incomplete_count: int (monsters with incomplete=True)
        - failures: list[ParseFailure] (action or structure failures)
        ImportResult.from_parse_result(filename: str, result: ParseResult) -> ImportResult:
          Classmethod that builds ImportResult from a ParseResult:
          success_count = len(result.monsters)
          incomplete_count = sum(1 for m in result.monsters if m.incomplete)
          failures = result.failures

      dispatch tests (inline fixture strings — not real files):
        - detect_format with 5etools content → 'fivetools'
        - detect_format with homebrewery content → 'homebrewery'
        - detect_format with plain content → 'plain'
        - detect_format with unrelated text → 'unknown'
        - parse_file with tmp_path fixture writing a minimal fivetools content file → returns ParseResult with monster
        - parse_file with nonexistent path → returns ParseResult with 1 failure, monsters=[]
        - ImportResult.from_parse_result with a 3-monster result (1 incomplete) → success_count=3, incomplete_count=1

    <implementation>
      src/parser/statblock_parser.py:
        Imports: re, pathlib.Path, src.parser.models.ParseResult/ParseFailure/ImportResult,
                 src.parser.formats.fivetools.parse_fivetools,
                 src.parser.formats.homebrewery.parse_homebrewery,
                 src.parser.formats.plain.parse_plain

        detect_format(content: str) -> str:
          Priority order (check 5etools first to avoid false positives):
          1. re.search(r'^>[-\s]+\*\*Armor Class\*\*', content, re.MULTILINE) → 'fivetools'
          2. re.search(r'^___\s*$', content, re.MULTILINE) and '**Armor Class**' in content → 'homebrewery'
          3. re.search(r'^##\s+\w', content, re.MULTILINE) and re.search(r'\*\*Armor Class\*\*', content) → 'plain'
          4. return 'unknown'

        parse_file(path: Path) -> ParseResult:
          try: content = path.read_text(encoding='utf-8')
          except (OSError, UnicodeDecodeError) as e:
            return ParseResult(monsters=[], failures=[ParseFailure(source_file=str(path), monster_name="", reason=str(e))], warnings=[])
          fmt = detect_format(content)
          if fmt == 'fivetools': return parse_fivetools(content)
          if fmt == 'homebrewery': return parse_homebrewery(content)
          if fmt == 'plain': return parse_plain(content)
          return ParseResult(monsters=[], failures=[], warnings=[f"Unrecognized format in {path.name}: skipped"])
