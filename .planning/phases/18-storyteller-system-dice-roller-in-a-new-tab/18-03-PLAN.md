**Settings persistence — restore path:**

`_apply_settings(self, settings: AppSettings)` (line 208) is the single method called when settings load or change. It is invoked:
- During `__init__` at line 150: `self._apply_settings(self._current_settings)`
- From `_on_settings_saved()` at line 199 when the user clicks Save in the Settings tab

Add the following block at the END of `_apply_settings()`, after the existing `ui_scale` block:

```python
# Restore storyteller system and last-used config
if hasattr(self, '_storyteller_tab'):
    self._storyteller_tab.restore_config(
        settings.storyteller_system,
        settings.storyteller_last_config,
    )
```

The `hasattr` guard is required because `_apply_settings` is called during `__init__` before `_storyteller_tab` is fully registered in `QTabWidget` (though the object exists). Without the guard, the call is safe; with it, it is defensive against future init reordering.

**`restore_config()` method in storyteller_tab.py:**

If `restore_config()` was not implemented in Plan 02 (or was left as a stub), implement it now. The ToggleBar programmatic selection API is `set_value(label: str)` — confirmed from `src/ui/toggle_bar.py`. It calls `_on_click` internally and emits `value_changed`, which triggers `_on_system_changed()` to show/hide panels.

```python
def restore_config(self, system: str, config: dict) -> None:
    label = "Aberrant 1e" if system == "aberrant" else "WoD Classic"
    self._system_toggle.set_value(label)   # ToggleBar.set_value() — emits value_changed
    self._wod_pool_spin.setValue(config.get("wod_pool", 5))
    self._wod_diff_spin.setValue(config.get("wod_difficulty", 6))
    self._wod_reroll_combo.setCurrentIndex(config.get("wod_reroll", 0))
    self._wod_rote_check.setChecked(config.get("wod_rote", False))
    self._ab_pool_spin.setValue(config.get("ab_pool", 5))
    self._ab_mega_spin.setValue(config.get("ab_mega", 0))
    self._ab_auto_spin.setValue(config.get("ab_auto", 0))
    self._ab_req_spin.setValue(config.get("ab_required", 1))
    self._name_edit.setText(config.get("character_name", ""))
```
