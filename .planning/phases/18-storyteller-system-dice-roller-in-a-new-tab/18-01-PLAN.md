**`src/storyteller/models.py`** — dataclasses only, no Qt:

```python
from __future__ import annotations
from dataclasses import dataclass, field

@dataclass
class DieResult:
    value: int
    is_success: bool
    is_one: bool
    is_reroll: bool = False  # True if added by 8-again/9-again/rote chain

@dataclass
class MegaDieResult:
    value: int
    sux_count: int           # 0, 2, or 3 (Aberrant mega rules)
    is_one: bool

@dataclass
class WodRollResult:
    dice: list[DieResult]    # all dice including all re-rolls, in order
    net_successes: int       # raw_successes - ones_count (can be negative)
    raw_successes: int       # count of dice with is_success=True
    ones_count: int          # count of non-success dice showing 1
    is_botch: bool           # net_successes < 0
    is_exceptional: bool     # net_successes >= 5
    reroll_threshold: int | None  # 8, 9, or None
    rote_enabled: bool
    pool: int
    difficulty: int

@dataclass
class AberrantRollResult:
    normal_dice: list[DieResult]
    mega_dice: list[MegaDieResult]
    auto_successes: int
    total_successes: int
    is_botch: bool           # total_successes==0 AND any die shows 1
    success_tier: str        # "none", "1-4", "5-8", "9-12", "13-16"
    pool: int
    mega_pool: int
    successes_required: int

@dataclass
class StorytellerPreset:
    name: str
    system: str              # "wod" or "aberrant"
    # WoD fields
    pool: int = 5
    difficulty: int = 6
    reroll_threshold: int | None = None
    rote_enabled: bool = False
    # Aberrant fields
    mega_pool: int = 0
    auto_successes: int = 0
    successes_required: int = 1

    def to_dict(self) -> dict:
        return {
            "name": self.name,
            "system": self.system,
            "pool": self.pool,
            "difficulty": self.difficulty,
            "reroll_threshold": self.reroll_threshold,
            "rote_enabled": self.rote_enabled,
            "mega_pool": self.mega_pool,
            "auto_successes": self.auto_successes,
            "successes_required": self.successes_required,
        }

    @classmethod
    def from_dict(cls, d: dict) -> "StorytellerPreset":
        return cls(
            name=d.get("name", ""),
            system=d.get("system", "wod"),
            pool=d.get("pool", 5),
            difficulty=d.get("difficulty", 6),
            reroll_threshold=d.get("reroll_threshold", None),
            rote_enabled=d.get("rote_enabled", False),
            mega_pool=d.get("mega_pool", 0),
            auto_successes=d.get("auto_successes", 0),
            successes_required=d.get("successes_required", 1),
        )
```

**`src/storyteller/engine.py`** — Qt-free, takes `random.Random` in constructor (matches Roller pattern):

```python
from __future__ import annotations
import random
from .models import DieResult, MegaDieResult, WodRollResult, AberrantRollResult

class StorytellerEngine:
    def __init__(self, rng: random.Random | None = None) -> None:
        self._rng = rng or random.Random()
```

**`roll_wod(pool, difficulty, reroll_threshold=None, rote_enabled=False)`:**
1. Roll `pool` d10s → initial `DieResult` list (is_reroll=False)
2. If `rote_enabled`: collect dice that are NOT success AND NOT showing 1 (pure failures), re-roll each once, append as is_reroll=True. After rote pass, qualifying dice (value >= reroll_threshold if threshold set) from the rote batch enter the 8/9-again chain below.
3. If `reroll_threshold` set: loop (max 50 iterations) — take the most recently added batch, find dice where value >= reroll_threshold, re-roll them, append as is_reroll=True, set them as the new "latest batch". Stop when no new qualifying dice.
4. Compute: raw_successes = count(is_success), ones_count = count(is_one AND NOT is_success), net_successes = raw - ones, is_botch = net < 0, is_exceptional = net >= 5.
5. Return WodRollResult with all fields.

Note: For rote + 8-again interaction — after the single rote pass, the rote-generated dice that meet the threshold enter the standard chain loop (rote dice become the initial batch for the loop).

**`roll_aberrant(pool, mega_pool, auto_successes, successes_required)`:**
1. Roll `pool` d10s → DieResult list (is_success = value >= 7, is_one = value == 1)
2. Roll `mega_pool` d10s → MegaDieResult list (sux_count: 10→3, 7-9→2, else 0; is_one = value == 1)
3. normal_sux = sum(1 for d in normal_dice if d.is_success)
4. mega_sux = sum(d.sux_count for d in mega_dice)
5. total = auto_successes + normal_sux + mega_sux
6. any_one = any die (normal or mega) with is_one=True
7. is_botch = (total == 0 and any_one)
8. tier: "none" if total==0 and not is_botch → actually use "none" only if total==0. Tiers: >=13→"13-16", >=9→"9-12", >=5→"5-8", else "1-4". If total==0, tier="none".
9. Return AberrantRollResult.

CRITICAL: Aberrant 1s never reduce successes. The botch check is purely (total==0 AND any_one). A roll of [7, 1, 1] with 0 mega → total = 1 success → NOT a botch.
